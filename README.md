# AI Steganography Project

This is my final year college project. It uses deep learning to hide data inside images and 3D files.

## Setup
1. Create a virtual environment: `python -m venv venv`
2. Activate it: `.\venv\Scripts\activate`
3. Install requirements: `pip install -r requirements.txt`
4. Run the app: `python app.py`

## Features
- Hide text, images, or 3D files inside other files
- AES-256 password encryption
- PyTorch based CNN architecture
- Swagger API documentation available at `/apidocs/`
